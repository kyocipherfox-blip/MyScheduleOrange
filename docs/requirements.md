| 形式 | 静的 Web アプリケーション（HTML / CSS / JavaScript） |
| 公開先 | GitHub Pages |

## 2. 目的

個人の日常スケジュールを視覚的に管理するための Web アプリを提供する。
インターネット接続がなくてもブラウザのローカルストレージにデータを保持し、
PC・スマートフォンの両方から快適に利用できることを目指す。

## 3. ステークホルダー・利用者

- 個人ユーザー（開発者自身を含む）
- デバイス：PC ブラウザ / スマートフォンブラウザ

## 4. ユーザーストーリー

| ID | ユーザーストーリー |
|----|------------------|
| US-01 | スケジュールを週単位の時間軸グリッドで確認したい |
| US-02 | 日付をクリックして新しい予定を素早く入力したい |
| US-03 | 予定をダブルクリックして内容を編集したい |

| US-06 | 前週・次週へ移動し、年間を通じた予定を管理したい |
| US-07 | 1週間表示と2週間表示を切り替えたい |
| US-08 | カテゴリの名前と色をカスタマイズしたい |
| US-09 | ページを再読み込みしてもデータが保持されてほしい |
| US-10 | スマートフォンで開いたとき画面全体を使ったフル画面表示で利用したい |

## 5. 機能要件

### 5.1 スケジュール表示
- 6:00〜22:00 の時間帯を縦軸に表示する
- 列は曜日（月〜日）対応し、1週間または2週間分を表示する
- 時間軸ラベルは選択間隔に応じて動的に表示する

### 5.2 予定の追加・編集・削除
- 日付列をクリック → モーダルで予定名・カテゴリ・開始/終了時間を入力して追加
- 繰り返し周期（なし/毎日/平日/毎週/毎月）と終了日を設定して繰り返し予定を一括登録
- 予定ブロックをダブルクリック → 同モーダルで編集・保存（その1件のみ変更）
- 予定ブロック右上の「×」ボタンで削除（繰り返し予定は1件のみ or 全件選択）

### 5.3 予定のリサイズ
- 予定ブロック下部のハンドルをドラッグして終了時間を変更

### 5.4 ナビゲーション
- 「◀ 前週」「次週 ▶」ボタンで週を切り替え
- 「1週間」「2週間」トグルで表示期間を変更

### 5.5 カテゴリ管理
- デフォルトカテゴリ：仕事・通勤 / 食事 / 運動 / レジャー・趣味
- カテゴリの追加・削除・名前変更・色変更が可能
- 削除されたカテゴリに属する予定はデフォルトカテゴリに移行

### 5.6 スケジュールグリッドのズーム
- ズームボタン（－ / ＋）でグリッドの表示倍率を 50%〜200% の範囲で変更できる
- ズーム変更後もクリック・ドラッグによる時刻計算が正確であること
- ズーム設定は localStorage に保存・復元される

### 5.7 データ永続化
- データはブラウザの localStorage に保存（キー：`schedule_v5`）
- 1年分相当（最大数千件）のデータを保持できること
- 保存内容：events / nextId / intervalIdx / viewMode / currentWeekStart / categories / zoomLevel
- デフォルト表示間隔：30分（intervalIdx = 1）

## 6. 非機能要件

| 区分 | 要件 |
|------|------|
| パフォーマンス | 500件のイベントがある場合でも描画が遅延なく完了すること |
| 互換性 | Chrome / Firefox / Safari / Edge の最新版で動作すること |
| レスポンシブ | 画面幅 360px 以上でスクロール可能なレイアウトを保つこと |
| セキュリティ | データは端末ローカルにのみ保存し、外部サーバーへ送信しないこと |
| 保守性 | CSS / JS / HTML を分離したファイル構造とし、ES Modules を使用すること |
| オフライン | インターネット接続がなくても全機能が利用できること |

## 7. 制約・前提

- バックエンドサーバーは存在しない（静的ファイルのみ）
- GitHub Pages でホスティングし、`index.html` がエントリポイント
- データは端末ごとに独立して管理される（同期機能は対象外）

## 8. 用語定義

| 用語 | 説明 |
|------|------|
| イベント / 予定 | スケジュール上の1件の予定データ |
| カテゴリ | 予定の種別（色・名前で管理） |
| 週スタート | 表示週の月曜日（YYYY-MM-DD） |
| viewMode | 表示週数（1 or 2） |
| intervalIdx | 時間軸間隔インデックス（0〜3）。デフォルトは1（30分） |
| recurringId | 繰り返し予定グループの共通 ID |
| zoomLevel | グリッド表示倍率（0.5〜2.0） |
